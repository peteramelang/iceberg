# iceberg

An interactive learning guide for production-readiness — the topics that sit "below the waterline" of the *Vibe Coding vs Production Reality* iceberg.

## What it is

Iceberg is a personal curriculum + companion web app for the engineering skills that distinguish a deployed MVP from a real production system: authentication, payments, idempotency, observability, scaling, incident response, compliance, and so on. 46 topics across 7 phases, with curated learning resources (videos, docs, services, courses) and a typed connection graph.

The content is generated by an autonomous multi-agent pipeline that proposes a taxonomy, researches each topic with parallel sub-agents, reconciles their picks, runs adversarial challenges, checks liveness, and maps relationships — committing after each step so the run is fully resumable.

## Quick start

```bash
npm install
npm run dev   # serves the app at http://localhost:5173
```

## Pipeline

The content pipeline is executed by Claude Code in a terminal session. Open this repo in Claude Code and say:

> continue the iceberg pipeline

Claude reads `pipeline/runbook.md` and resumes from `content/_ledger.json`. See `pipeline/README.md` for the runbook and stage prompts.
